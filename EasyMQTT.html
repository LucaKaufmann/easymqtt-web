Features

Everything you need from an MQTT client, nothing you don't.

EasyMQTT ships the essentials of a great MQTT client — connect, subscribe, publish — plus the iOS‑native touches that make it a joy to use every day.

Connect

Connect to any MQTT broker

Plug in a URL, username, and password — or scan a recent broker on your network. Save multiple brokers, switch in a tap, and pick MQTT 3.1.1 or 5.0.

Subscribe

Support

Stuck? Here's every answer.

The common questions, step‑by‑step tutorials, and a quick line to the developer.

The app won't connect to my broker+
First, try the public broker broker.hivemq.com to confirm the app can reach the internet. If that works, double‑check your host, port, and credentials — most connection failures come from a wrong port (1883 vs 8883 for TLS) or a stale client ID.
How do I configure Siri Shortcuts?+
Open the Shortcuts app, tap +, and search for EasyMQTT. You'll see actions for Send Message and Fetch Message. Pick a broker, type a topic, and either hardcode a payload or use a previous action's output.
I can't see any brokers when setting up a Shortcut or Widget+
Make sure the broker is saved in the app — either via Add to favorites on the Connect screen, or under Settings → Brokers. Give it a name and save. The broker should now appear in the Shortcuts and Widgets pickers.
What do the Send Message and Fetch Message shortcuts do?+
Send Message publishes a payload to a topic on a broker — with optional retain and QoS. Fetch Message subscribes to a topic and returns the most recent retained message, handy for reading state into other Shortcut actions.
What types of widgets are there?+
Three kinds: Single value (one topic, one number), Dual value (two topics side‑by‑side), and Graph widget (a sparkline from a topic's history). All refresh in the background every few minutes.
Does EasyMQTT support TLS / SSL?+
Yes — tick the TLS/SSL toggle on the Connect screen. EasyMQTT uses the system trust store; self‑signed certs aren't supported yet.
Is my data synced across devices?+
Brokers, favorites, and locations sync via iCloud if you're signed in. Messages are kept locally — they never leave your device.
Does EasyMQTT run on Mac?+
Yes — the Mac Catalyst build is on the Mac App Store. Same UI, keyboard shortcuts, and menu bar integration for quick publishes.
